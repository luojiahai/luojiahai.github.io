# 叉烧

## 材料

| 材料   | 用量    |
| ------ | ------- |
| 梅花肉 | 1000 克 |
| 生抽   | 25 克   |
| 白砂糖 | 40 克   |
| 料酒   | 20 克   |
| 蚝油   | 15 克   |
| 红腐乳 | 1 块    |

## 制作步骤

1. 将生抽、白砂糖、料酒、蚝油、红腐乳在碗中搅拌均匀，制成叉烧酱。
2. 梅花肉切片，在密封盒中倒入叉烧酱将肉淹没，放入冰箱腌制 5 小时。
3. 将腌制后的梅花肉送入预热到 200°c 的烤箱，并每隔 10 分钟取出烤箱再刷一遍叉烧酱，反复 2 次，共计 30 分钟。
