# 布朗尼

## 材料

| 材料     | 用量    |
| -------- | ------- |
| 黄油     | 230 克  |
| 白砂糖   | 230 克  |
| 可可粉   | 50 克   |
| 鸡蛋     | 3 个    |
| 香草精   | 10 毫升 |
| 盐       | 少许    |
| 面粉     | 120 克  |
| 巧克力豆 | 适量    |

## 制作步骤

1. 将烤箱预热至 180°C（356°F）。
2. 加热融化黄油直至非常热。
3. 将黄油、白砂糖和可可粉倒入碗中，大力搅拌直至糖完全溶解。
4. 加入鸡蛋、香草精和盐，继续搅拌至顺滑。
5. 加入面粉和巧克力豆，搅拌至面粉完全混合。
6. 将面糊倒入烤盘。
7. 在烤箱中烘烤 25 分钟。
