# 布朗尼

## 材料

| 材料           | 用量    |
| -------------- | ------- |
| 黄油           | 226 克  |
| 白砂糖         | 200 克  |
| 荷兰可可粉     | 50 克   |
| 意式浓缩咖啡粉 | 10 克   |
| 鸡蛋           | 3 个    |
| 香草精         | 10 毫升 |
| 盐             | 少许    |
| 面粉           | 100 克  |
| 巧克力豆       | 适量    |

## 制作步骤

1. 将烤箱预热至 180°c（356°f）。
2. 加热融化黄油。
3. 将黄油、白砂糖、可可粉和意式浓缩咖啡粉倒入碗中，大力搅拌直至糖完全溶解。
4. 加入鸡蛋、香草精和盐，继续搅拌至顺滑。
5. 加入面粉和巧克力豆，搅拌至面粉完全混合。
6. 将面糊倒入烤盘。
7. 在烤箱中烘烤 25 分钟。
