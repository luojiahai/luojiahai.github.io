# 生滚粥

## 材料

| 材料           | 用量      |
| -------------- | --------- |
| 任意肉类或海鲜 | 随意      |
| 任意蔬菜       | 随意      |
| 大米           | 100g      |
| 水             | 1000 毫升 |
| 白胡椒         | 适量      |
| 鱼露           | 适量      |
| 盐             | 适量      |

## 制作步骤

1. 将大米淘洗干净后，把大米和水一起放入电饭煲中，选择煮粥模式煮约 40 分钟，不需要煮太烂。
2. 将肉类或海鲜切成薄片。
3. 将蔬菜切碎。
4. 粥煮好后，将其倒入大锅中，开大火加热至沸腾。
5. 粥沸腾后，放入切好的肉类或海鲜，用筷子轻轻搅拌防止粘底。
6. 待肉类变色断生后，放入切好的蔬菜碎。
7. 待所有食材煮熟、粥重新沸腾后，加入适量盐、鱼露和白胡椒粉调味，搅拌均匀即可关火上桌。
