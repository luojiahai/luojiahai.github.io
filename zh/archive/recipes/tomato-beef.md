# 番茄牛肉

## 材料

| 材料   | 用量   |
| ------ | ------ |
| 牛肉   | 500 克 |
| 番茄   | 3 个   |
| 洋葱   | 1 个   |
| 姜     | 适量   |
| 蒜     | 适量   |
| 生抽   | 10 克  |
| 蚝油   | 10 克  |
| 料酒   | 10 克  |
| 胡椒粉 | 适量   |
| 番茄酱 | 10 克  |
| 白砂糖 | 5 克   |
| 食用油 | 适量   |

## 制作步骤

1. 牛肉切薄片，用生抽、蚝油、料酒腌制 30 分钟。
2. 番茄洗净切块，洋葱切块，姜切片，蒜切片。
3. 热锅倒油，放入姜片和蒜片爆香。
4. 放入牛肉片大火翻炒至变色。
5. 加入番茄块翻炒，炒至番茄出汁。
6. 加入洋葱块继续翻炒。
7. 加入番茄酱、蚝油、白砂糖调味。
