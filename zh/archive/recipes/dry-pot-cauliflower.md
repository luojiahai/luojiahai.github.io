# 干锅花菜

## 材料

| 材料           | 用量  |
| -------------- | ----- |
| 花菜           | 1 棵  |
| 五花肉（可选） | 100g  |
| 生姜           | 适量  |
| 大蒜           | 适量  |
| 料酒           | 5 克  |
| 花椒           | 5 克  |
| 干辣椒         | 适量  |
| 生抽           | 10 克 |
| 盐             | 适量  |
| 黑胡椒粉       | 适量  |

## 制作步骤

1. 花菜洗净，切小朵。
2. 五花肉切薄片。
3. 生姜切片，大蒜切末。
4. 热锅温油，放入姜片和蒜末翻炒，再加入五花肉片煸炒，添料酒、花椒煸香。
5. 待五花肉的油被超出后，加入花菜翻炒。
6. 添干辣椒、生抽、盐、黑胡椒粉继续翻炒，直至花菜的水分被炒干即可。
