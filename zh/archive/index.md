# 索引

这是我的个人网站，使用 [vitepress](https://vitepress.dev/) 静态生成，部署在 [github pages](https://docs.github.com/en/pages) 上。网站提供英文和简体中文两种语言，支持浅色和深色主题。

这个网站的图标是一张 💾 软盘，我用过的第一种便携式存储设备，就是 3½ 英寸的那种。那时的电脑几乎都内置软盘驱动器。

我的网名 `luojiahai` 是我名字罗嘉海的拼音。在各种账号、邮箱和社交媒体上，我都用这个网名。在现实生活中，在英语地区，我用英文名 geoffrey，发音跟 jeffrey 一样，不是开头读成 jee-oh。
